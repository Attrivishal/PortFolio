# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest   | ✅ Yes    |
| < latest | ❌ No     |

## Reporting a Vulnerability

If you discover a security vulnerability in my portfolio, please:

1. **Do NOT** open a public GitHub issue
2. Email me directly at: vishalattri196@gmail.com
3. Include detailed steps to reproduce the issue

I will:
- Acknowledge receipt within 48 hours
- Investigate and provide a timeline for fix
- Credit you in the release notes (if you wish)

## Responsible Disclosure

I follow responsible disclosure practices. Please give me 14 days to fix the issue before public disclosure.

## Security Update Process

- Security fixes are automatically deployed via CI/CD
- New Docker images are pushed to Docker Hub with `:latest` tag
- Check the [Docker Hub release notes](https://hub.docker.com/r/vishalattri/portfolio) for security updates
